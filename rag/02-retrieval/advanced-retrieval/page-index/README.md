# PageIndex

长文档层级树索引与推理导航检索框架（VectifyAI）。

## 定位

本目录收录对 PageIndex 开源检索框架的研究。PageIndex 是一个具体的工程对象，不是泛化范式名。

它在此处与 hybrid-retrieval、multi-hop-retrieval、recursive-retrieval 等并列，作为"高级检索"路线的一种具体对象研究。
