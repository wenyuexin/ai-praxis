# Delegation and Control（委托与控制）

本目录用于沉淀用户如何向 Agent 委托任务、约束执行范围、控制权限与管理自动化程度。

- `delegation-granularity.md`：讨论用户委托的是目标、计划、步骤、动作、能力边界还是整条工作流。
- `control-surface-design.md`：讨论 approval point、autonomy budget、permission boundary、stop condition、takeover / fallback entry 等控制点如何被暴露和操作。
