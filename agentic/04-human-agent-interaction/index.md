# 04 Human-Agent Interaction Index

> 如果你已经知道自己要找的是人机协作中的哪类委托、控制、交互或信任问题，从这里开始。
> 边界：本文件负责目录结构展示与查找导航，不替代 `README.md` 的目录定位说明，也不替代 `overview.md` 的整体理解主线。

## 目录结构

```text
04-human-agent-interaction/
├── human-in-the-loop/        # 人在回路与人工介入
├── delegation-and-control/   # 任务委托、权限与控制面
├── interaction-surfaces/     # 交互表面、状态呈现与操作入口
├── trust-and-alignment/      # 信任、预期与偏好对齐
├── overview.md               # 人机交互整体理解主线
└── backlog.md                # 当前内容缺口
```

## 按问题找入口

- 想看人工确认、人工审核、人工接管或人在回路：先看 `human-in-the-loop/`
- 想看任务委托、自治程度、权限边界或控制权设计：先看 `delegation-and-control/`
- 想看 Agent 面向用户的界面、状态展示、操作入口或反馈机制：先看 `interaction-surfaces/`
- 想看信任建立、行为预期、解释边界或偏好对齐：先看 `trust-and-alignment/`

## 按阅读目标找入口

- 想先恢复人机交互的整体理解主线：读 [`overview.md`](./overview.md)
