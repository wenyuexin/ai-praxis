# OpenCode Research Notes

> 本目录承接 OpenCode 研究过程材料，包括源码核验、官方文档摘录、证据对照、失败搜索和未闭合问题。这里是研究辅助材料入口，不是面向读者的对象正文。

## 目录结构

```text
notes/
```

## 使用边界

- `source.md`：记录源码和官方文档核验过程，包括关键路径、搜索线索、阶段性观察、失败搜索和待查问题。
- `evidence.md`：整理 claim-source 对照、Evidence 状态和可回流正文的证据。
- `runtime-validation.md`：承接动态运行补证计划、验证矩阵和后续运行记录。
- 不在本目录直接写对象级结论；稳定结论已经或应继续回流到案例正文或机制专题，并保留 Evidence / Trace。
