# Codex Agent（代码能力）

围绕 **Codex 作为代码 Agent 本体** 的系列文档，聚焦原理、使用方法与证据核验。

## 目录结构

```text
codex/
├── README.md                   # 目录导览（L1 定向）
├── codex-agent-overview.md     # 定义、边界、定位
├── codex-agent-mechanisms.md   # 原理、闭环、上下文与工具机制
├── codex-agent-practice.md     # 使用方法、任务范式与反模式
├── codex-agent-evidence.md     # 证据分级、关键事实与时间轴
└── tmp.md                      # 调研原始素材（持续更新）
```

## 阅读顺序

1. `codex-agent-overview.md`
2. `codex-agent-mechanisms.md`
3. `codex-agent-practice.md`
4. `codex-agent-evidence.md`

## 写作与维护规则

- 新调研先写入 `tmp.md`，再并入 4 篇主文档。
- 主文档只保留“机制、边界、方法、证据”，避免碎片堆砌。
- 每次更新补充“版本变更记录”，保持可追溯。
