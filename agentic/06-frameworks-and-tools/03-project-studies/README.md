# Project Studies（项目案例研究）

本目录用于沉淀完整 Agent 系统或开源项目的系统化拆解，关注真实项目如何组合 agentic 各层能力并落地为工程系统。

## 收录范围

适合放入本目录的对象：

- 完整 Agent 系统
- 自托管 Agent 平台
- 具有独立架构、运行时、工具系统、记忆系统或网关系统的开源项目
- 值得系统拆解的产品化项目案例

当前案例：

- `hermes-agent/`
- `openclaw/`

## 与框架目录的区别

- `01-frameworks/` 关注“可复用框架 / SDK / 编排平台”。
- `03-project-studies/` 关注“一个完整系统如何把框架、工具、记忆、上下文、网关、安全、部署等能力组装起来”。

因此，Hermes Agent、OpenClaw 这类对象更适合作为项目案例，而不是底层能力目录或通用框架目录。
