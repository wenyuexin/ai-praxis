# Organizational

> 适用范围：多智能体组织结构与层级治理研究
> 阶段状态：调研优先，暂不收敛实现方案

## 一句话定位

本目录用于整理多智能体系统中的组织结构设计，包括层级关系、角色制度、团队形态与治理机制。

## 快速入口

- 结合上层 `../README.md` 理解 organizational 与 collaboration、coordination 的差异。
- 重点关注层级组织、角色制度、中央控制与自治平衡、治理成本与演化机制。
- 如出现组织模型、术语定义或版本口径冲突，记录到 `../../temp/conflict.md`。

## 边界说明

- 放在这里：层级结构、团队形态、角色制度、治理机制、中央化与去中心化权衡。
- 不放在这里：具体通信协议（放 `../communication-protocols/`）、任务级调度（放 `../coordination/`）、共享状态存储（放 `../shared-memory/`）。

## 同目录导航

- 建议结合 `../collaboration/README.md` 理解角色分工如何上升为组织层级与制度设计。
- 建议结合 `../coordination/README.md` 对比组织治理与任务级调度之间的边界。
