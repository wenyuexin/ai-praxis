# Agent System Modeling（智能体系统建模）

本目录用于沉淀 Agent 系统的总体模型、分层框架与信息流抽象，帮助连接基础定义、认知架构与后续工程实现。

## 入口说明

- 第一次进入时，先用本页判断你的问题是否属于 Agent 系统整体建模，而不是某个具体框架、任务流或实现细节
- 如果你当前更需要的是下一步该跟踪哪些论文、survey 或候选对象，继续转到 [`candidates.md`](./candidates.md)

## 当前目录定位

这里优先承接 foundations 层的系统建模问题，例如：

- Agent harness / runtime substrate 是否可以上升为系统建模对象
- 认知架构、编排架构与运行时基座之间如何分层
- 从论文、survey 与对象研究中抽出更稳定的上位概念时，应落在哪个建模层

## 相邻边界

- 具体框架、产品或工程工具的实现细节，不放在这里，优先回到对应主题目录
- foundations 层的一般问题缺口由 `../../01-foundations/backlog.md` 承接；本目录只在对象队列先稳定下来时，用 `candidates.md` 跟踪下一步值得研究的对象
- 当这里已经形成足够稳定的上位判断，再考虑继续沉淀为正式专题或总论；在那之前，不急着补 `overview.md`
