# 学习资料 (Learning Materials)

本目录用于汇总学习相关资源，优先按资源类型以单文件形式收纳，作为正式知识目录之外的辅助入口。

## 入口说明

- 第一次进入时，先用本页判断你要找的是哪类学习资源
- 如果你要看尚未沉淀进正式知识目录的待补主题与方向缺口，继续转到 [`backlog.md`](./backlog.md)

## 目录定位

这里主要承接几类辅助材料：

- 书籍、课程、论文、博客、工具、数据集、社区等外部学习资源
- `interview/` 下的面试相关资料
- `scratch/` 下的零散记录、临时摘记与后续可回流内容

## 相邻边界

- 已经形成稳定主题判断、适合进入正式知识体系的内容，应回到对应领域目录
- 仍处于资料收集、方向补缺或零散记录阶段的内容，优先留在本目录
