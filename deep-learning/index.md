# Deep Learning Index

> 如果你已经知道自己要找的是哪类深度学习主题或子目录，从这里开始。
> 边界：本文件负责目录结构展示与子方向定位，不替代 `README.md` 的目录说明，也不替代整体理解主文或学习路径文档。

## 目录结构

```text
deep-learning/
├── 01-neural-network-fundamentals/  # 神经网络基础
├── 02-training-and-optimization/    # 训练与优化
├── 03-architectures/                # 按领域划分的主流架构
├── 04-advanced-topics/              # 特定方向的进阶方法论
└── 05-infra/                        # 深度学习基础设施
```

## 按子目录定位

- `01-neural-network-fundamentals/`：神经元、激活函数、损失函数等最底层概念
- `02-training-and-optimization/`：反向传播、优化器、正则化与初始化方法
- `03-architectures/`：CNN、RNN、Transformer 与生成模型等主流架构
- `04-advanced-topics/`：自监督学习、元学习、NAS、持续学习等进阶主题
- `05-infra/`：GPU 计算、分布式训练与框架对比等工程支撑
